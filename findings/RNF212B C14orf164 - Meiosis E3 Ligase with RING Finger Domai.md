# RNF212B (C14orf164) - Meiosis E3 Ligase with RING Finger Domain

**Date:** 2026-03-16T14:03:05.276626

**Quality Score:** 4/10 (MODERATE)

## Description
RNF212B is a 300aa E3 ubiquitin-protein ligase containing a C4HC3 RING finger domain (aa 4-40). InterPro classifies it as Zip3/RNF212-like family with SUMO transferase activity. STRING interactions confirm meiotic function: partners include HFM1, MSH4, REC114, REC8, SPO11, TEX11, MLH3 - all critical for meiotic recombination and synaptonemal complex formation. HPA shows kidney-enriched expression (24.2 nTPM) with nuclear/nucleolar localization. This gene is essential for crossing-over formation during meiosis prophase I.
