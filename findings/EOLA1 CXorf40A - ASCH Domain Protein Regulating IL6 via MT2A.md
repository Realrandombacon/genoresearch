# EOLA1 (CXorf40A) - ASCH Domain Protein Regulating IL6 via MT2A Interaction

**Date:** 2026-03-16T12:03:22.465666

**Quality Score:** 6/10 (GOOD)

## Description
158aa X-chromosome protein containing conserved ASCH domain (aa 6-114) and PUA-like superfamily fold. Originally annotated as CXorf40A (chromosome X open reading frame). Functions in cell protection during inflammatory response - negatively regulates IL6 production and apoptosis through MT2A (metallothionein-2A) regulation. High-confidence STRING interaction with MT2A (0.826 score) supports functional link. Ubiquitous tissue expression with cell-type enhanced specificity. Conserved across primates (96% identity in orangutan). Paralog EOLA2 exists (98.7% identity). Represents partially characterized dark gene with defined molecular mechanism.
