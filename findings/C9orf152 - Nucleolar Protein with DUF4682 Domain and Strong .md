# C9orf152 - Nucleolar Protein with DUF4682 Domain and Strong Dosage Sensitivity

## Description
C9orf152 encodes a 239 aa uncharacterized nucleolar protein containing a conserved DUF4682 domain (PF15733, IPR032738) homologous to the C-terminal region of TBC1 domain family member 30. DUF4682 is a domain of unknown function found exclusively in this protein family. Critical functional insights: (1) STRING reveals interaction network including SMKR1 (splicing modulator), RAB30 (vesicle trafficking GTPase), and IL10RA (cytokine receptor) — suggests roles in RNA processing and membrane trafficking; (2) HPA shows nucleoplasm/nucleoli localization with intestine-enriched expression (39.1 nTPM) and cancer-enhanced pattern; (3) 34 pathogenic ClinVar CNVs (both copy number losses and gains) demonstrate extreme dosage sensitivity, confirming essential biological function; (4) BLAST reveals human-specific sequence with no cross-species conservation. Function hypothesis: C9orf152 is a nucleolar scaffold protein that coordinates ribosomal RNA processing or ribosome biogenesis via the DUF4682 domain; interacts with splicing factors (SMKR1) and trafficking machinery (RAB30) to couple transcription/processing with nuclear export. Dosage imbalance disrupts nucleolar organization and ribosome production, contributing to intestinal epithelial dysfunction and cancer progression. The DUF4682 domain represents an uncharacterized protein-protein interaction module.
