# ERCC6L2: SNF2 Helicase with DNA Repair Function and 117 Pathogenic Variants

**Date:** 2026-03-18T06:35:01.944779

**Quality Score:** 9/10 (EXCELLENT)

## Description
ERCC6L2 (C9orf102) encodes a 1561 amino acid DNA excision repair protein with multiple helicase domains. InterPro analysis reveals DEXH-box helicase domain (aa 134-382), SNF2 N-terminal domain (aa 137-469), C-terminal helicase domain (aa 520-673), Tudor domain (aa 29-84), and AAA-ATPase domains. The protein functions in double-strand break end-joining and programmed recombination. STRING interactions show medium-confidence associations with DNA repair proteins: FANCM (0.624), WRN (0.614), LRGUK (0.576), ELOF1 (0.568), RAD51 (0.491), and SAMD9L (0.519). HPA shows ubiquitous expression across all tissues with low specificity, localized to nucleoplasm and cytosol. ClinVar contains 117 pathogenic/likely pathogenic variants including frameshift deletions (c.2590del, c.3762del) and nonsense mutations (p.Arg97Ter, p.Gln574Ter). AlphaFold structure (pLDDT 57.8) indicates medium confidence with partially ordered domains and disordered regions, consistent with a large multi-domain helicase. Conservation data shows homologs in Xenopus tropicalis (1397 aa, ~89% identity in catalytic core).
