# RNF213 (C17orf27): Giant E3 Ubiquitin Ligase with AAA+ ATPase Domains and Angiogenesis Function

**Date:** 2026-03-18T02:55:30.006911

**Quality Score:** 10/10 (EXCELLENT)

## Description
RNF213 (formerly C17orf27) encodes a massive 5207 aa E3 ubiquitin-protein ligase (UniProt Q63HN8) with multiple functional domains. InterPro analysis reveals: RING-type zinc finger (IPR001841, IPR018957) at residues 3997-4035 for E3 ligase activity, two AAA+ ATPase domains (IPR003593) at residues 2412-2557 and 2761-2910 for ATP-dependent remodeling, and a P-loop NTPase superfamily (IPR027417). AlphaFold predicts high-confidence structure (pLDDT 86.2) for the N-terminal region (1-557 aa). HPA shows ubiquitous expression across all tissues with cytosolic localization, classified as cancer-related and disease-associated. STRING reveals high-confidence interactions with MYH9 (0.892, cytoskeletal motor), UBE2L5 (0.809, E2 ubiquitin conjugating enzyme), PARP14/15 (0.706/0.677, ADP-ribosyltransferases), and UBE2L6 (0.607), suggesting roles in ubiquitination and cytoskeletal regulation. ClinVar contains 161 pathogenic variants including missense (p.Arg4062Trp, p.Val3121Ile) and truncating mutations. Orthologs in mouse (5148 aa), zebrafish (5061-5209 aa) indicate strong vertebrate conservation. FUNCTIONAL HYPOTHESIS: RNF213 functions as a dual E3 ubiquitin ligase and AAA+ ATPase, regulating protein turnover and cytoskeletal dynamics in endothelial cells. MECHANISTIC PROPOSAL: RING domain recruits E2 ubiquitin conjugating enzymes (UBE2L5/6) for substrate ubiquitination; AAA+ domains provide ATP-dependent conformational changes for substrate remodeling. Interaction with MYH9 suggests cytoskeletal coordination. Pathogenic variants cluster in functional domains, disrupting ubiquitination and angiogenesis pathways. NOTE: This gene is NOT truly "dark" - it is well-studied as Moyamoya disease susceptibility gene.
