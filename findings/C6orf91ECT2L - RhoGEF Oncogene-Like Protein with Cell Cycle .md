# C6orf91/ECT2L - RhoGEF Oncogene-Like Protein with Cell Cycle Regulation

**Date:** 2026-03-16T19:25:25.851302

**Quality Score:** 10/10 (EXCELLENT)

## Description
C6orf91 (ECT2L) encodes a 904-amino acid epithelial cell-transforming sequence 2 oncogene-like protein (UniProt Q008S8) functioning as a guanine nucleotide exchange factor (GEF) for Rho GTPases. InterPro analysis reveals multiple functional domains: RhoGEF/cdc25 homology domain (cd00160/IPR000219, aa 567-752) catalyzing nucleotide exchange, F-box domain (cd22173/IPR001810, aa 76-127) mediating ubiquitin ligase substrate recognition, Dbl homology domain (G3DSA:1.20.900.10, aa 559-765) for GTPase activation, Pleckstrin homology domain (G3DSA:2.30.29.30, aa 766-904) for membrane targeting, and DUF4347 (IPR025592, aa 291-459) of unknown function. STRING interaction network shows high-confidence binding to cell cycle regulators: CCDC28A (0.777), condensin subunit NCAPG (0.623), RACGAP1 (0.604) for cytokinesis, SPAG6 (0.599), CFAP161 (0.594), FLACC1 (0.594), F-box proteins FBXO47 (0.586) and FBXO48 (0.566), mitotic kinase TTK (0.580), and GSTM3 (0.544). HPA data indicates fallopian tube-enriched expression (8.1 nTPM), tissue-enriched pattern, nucleoplasmic and vesicular localization, classified as guanine-nucleotide releasing factor. ClinVar contains 15 pathogenic/likely pathogenic variants including copy number gains/losses at 6q23-24. AlphaFold structure (pLDDT 74.5) confirms well-folded multi-domain architecture. ECT2L likely functions as a dual regulator coordinating Rho GTPase signaling with SCF ubiquitin ligase-mediated protein degradation during cell division, with the F-box domain recruiting substrates for ubiquitination while the RhoGEF domain activates cytoskeletal remodeling. Fallopian tube enrichment suggests specialized roles in epithelial cell proliferation and ciliogenesis.
