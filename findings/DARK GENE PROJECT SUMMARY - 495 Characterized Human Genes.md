## Description
The dark gene characterization project has systematically analyzed 495 understudied human protein-coding genes from the ~17,000 "dark genome" (genes beyond the ~2,000 well-studied). Each finding integrates multi-omics evidence: InterPro/Pfam domains (IPR/PF accessions), STRING protein interactions (partner names + confidence scores), HPA tissue expression (tissue names + nTPM values), ClinVar pathogenic variants (counts + disease associations), AlphaFold structures (pLDDT scores), and cross-species conservation (% identity with mouse/zebrafish). High-quality findings (9-10/10) like TMEM253 demonstrate rich cross-domain reasoning: 217aa intestine-enriched transmembrane protein (91.3 nTPM) with 30 ClinVar pathogenic variants, interacting with WDR49/NOXRED1/ARFGEF3 (STRING scores 0.41-0.58), suggesting roles in intestinal epithelial function and vesicle trafficking. Other notable findings include C3orf62 (spermatogenesis factor with EP300/CREBBP chromatin modifier interactions), C10orf105 (brain-enriched DUF5527 protein with immune/cancer expression), and ANKMY1 (testis-enriched chromatin regulator). Mechanistic proposals link protein structure, subcellular localization, interaction networks, and disease variants into testable functional hypotheses. This resource transforms uncharacterized genomic "dark matter" into prioritized targets for experimental validation.
