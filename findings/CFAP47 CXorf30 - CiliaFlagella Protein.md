# CFAP47 (CXorf30) - Cilia/Flagella Protein

**Date:** 2026-03-16T10:12:28.661457

**Quality Score:** 5/10 (GOOD)

## Description
Former dark gene CXorf30 now characterized as CFAP47. Large protein (3187 aa) with multiple immunoglobulin-like domains and calponin homology domain. Functions in flagellar formation and sperm motility. Interacts with cilia/flagella protein network (TTC29, CFAP74, CFAP43, CFAP65). Expressed in fallopian tube, pituitary, retina. Cancer-enhanced expression.
