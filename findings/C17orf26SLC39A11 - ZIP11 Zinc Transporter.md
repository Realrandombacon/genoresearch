# C17orf26/SLC39A11 - ZIP11 Zinc Transporter

**Date:** 2026-03-16T15:06:34.796134

**Quality Score:** 3/10 (MODERATE)

## Description
Formerly annotated as dark gene C17orf26, this is actually SLC39A11 encoding ZIP11 zinc transporter. Contains ZIP zinc transporter domain (PF02535) spanning aa 26-337. Functions as zinc importer regulating cytosolic zinc concentrations. Expressed ubiquitously across all tissues with low specificity. Localized to nucleoplasm. Not a true dark gene - well-characterized transporter.
